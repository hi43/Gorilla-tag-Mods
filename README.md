## 🪐 Planet Destroyer (bonus game)

A browser game in [`planet-destroyer/`](planet-destroyer/) — fly a ship around a solar system and blow up every planet with an arsenal of 8 gadgets (pulse laser, homing missiles, giga beam, tesla arc, asteroid storm, core cracker, antimatter bomb, black hole). Earn cosmic dust, buy upgrades, warp to tougher systems.

**To play:** download `planet-destroyer/index.html` and open it in any browser. No install needed.
