⚡ Task Solver AI

Paste one task or a whole list — writing, planning, summarizing, researching, problem-solving — and it works through them all. Attach files from your computer (PDFs, photos, documents) and save the results back when it's done.

🔑 Setup — Claude API key
Get a key at console.anthropic.com. The key is saved to this browser's localStorage and sent only to Anthropic's API — never anywhere else.

Controls how simple or advanced the writing is — handy for writing to a younger reader, someone learning English, or explaining things simply. It's just a writing style; it doesn't disguise anything.

Pick a folder on your computer — it reads every assignment file inside (text, PDFs, photos), does the work, and saves the finished results into a done folder it creates right there. Works in Chrome/Edge on a computer; on phones use 📎/📷 on a task instead.

Tip: press Ctrl/⌘ + Enter to run.
